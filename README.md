# `tig`

`tig` is a Firefox extension that allows you to tag pages with a color and some text.

`tig` may be useful for tagging pages in:

- Large documentation sites as:
  - Read (green)
  - Reading (yellow)
  - Unread (red)
- Revision sites as:
  - Mastered (green)
  - Learnt (yellow)
  - Need Help (orange)
  - Not Learnt (red)

## Development

### Building

$ npm init
$ npm install
$ npm run build
